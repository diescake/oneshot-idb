# oneshot-idb

A wrapper library to easy to handle indexedDB for saving one shot data.

## How to use

t.b.d.

## Requirements

- Node.js v10.17.0
  - Probably the newer version is also fine.
- Recommend to use version manager [asdf](https://github.com/asdf-vm/asdf).
